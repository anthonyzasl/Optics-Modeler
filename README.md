# Optics-Modeler

## 1. Загальні відомості
Цей репозиторій містить вихідний код курсового проєкту, розробленого як рішення (Solution) для середовища Visual Studio.

* **Назва рішення:** kursova 
* **Тип проєкту:** C# (Visual C# Project) 
* **Основний проєкт:** `kursova\kursova.csproj` 

---

## 2. Технічні характеристики розробки
Для розробки та збірки проєкту використовувалися наступні інструменти:

| Параметр | Значення |
| :--- | :--- |
| **Середовище розробки** | Microsoft Visual Studio (v17) |
| **Версія рішення (Format)** | 12.00 |
| **Мінімальна версія VS** | 10.0.40219.1 |
| **Платформа** | Any CPU (Debug/Release) |

---

## 3. Структура проєкту
Проєкт організований за стандартною схемою Visual Studio:
* **`kursova.sln`** — головний файл рішення, що об'єднує всі компоненти.
* **`kursova/`** — папка, що містить основний вихідний код, ресурси та логіку програми.

---

## 4. Вимоги до системи та запуск
Для успішного відкриття та компіляції проєкту необхідно:

1. **Встановлене ПЗ:** Microsoft Visual Studio 2022 (версія 17.6 або новіша).
2. **Компоненти:** Робоче навантаження ".NET desktop development" (або відповідне до типу вашого проєкту).

### Інструкція із запуску:
1. Завантажте або склонуйте репозиторій.
2. Відкрийте файл **`kursova.sln`** через Visual Studio.
3. Виберіть конфігурацію збірки (**Debug** або **Release**).
4. Натисніть **F5** або кнопку **Start** для компіляції та запуску програми.
